# KIBO project

It consists of a panel of 5x5 addressable leds that could be configured through bluetooth.

# BOM

* E73-2G4M08S1C: MCU Module with nRF52840
https://www.fr-ebyte.com/products/E73-2G4M08S1C

* WS2812B-2020: Leds x25
https://docs.nordicsemi.com/bundle/ncs-2.6.1/page/zephyr/samples/drivers/led_ws2812/README.html

* MT3608: Step Up Converter 3.3V -> 5V
https://www.olimex.com/Products/Breadboarding/BB-PWR-3608/resources/MT3608.pdf

* AP2112K-3.3: LDO regulator
https://www.diodes.com/assets/Datasheets/AP2112.pdf

* TP4057: Charger for battery
http://toppwr.com/uploadfile/file/20230304/640302a47b738.pdf
